Baidu Submit URL是一款适用于站长主动向Baidu提交网站新增、更新和删除内容URL的WP插件。

== Description ==

Baidu Submit URLs插件是基于百度站长平台对站长开放的链接提交接口开发的，支持站长通过百度站长平台申请获取转入密钥，向百度搜索引擎主动推送链接，提升百度搜索引擎对网站的收录效率。


在使用Baidu Submit URL插件前，你首先需要到 “WordPress 后台-插件-已安装的插件”，找到插件 Baidu Submit URL，点击启动；然后点击“设置”或者通过 “WordPress 后台-插件-Baidu Submit URLs”对插件进行配置：

该插件的主要目的在于，大大提升原创内容网站数据被百度搜索引擎收录的效率，对网站内容起到一定的保护作用。


== Installation ==

1. 上传 `baidu-submit-url` 目录到 `/wp-content/plugins/`目录.
2. 在 WordPress 插件面板激活 `Baidu Submit URL` 插件.
3. 通过设置->`Baidu-Submit-URL` 配置准入密钥，即可向百度实时推送网站链接.
